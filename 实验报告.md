## 1.1 实验原理

### 1.1.1 SSM 框架概述

SSM 框架是 Spring + Spring MVC + MyBatis 三个开源框架的整合，是目前 Java Web 开发中广泛使用的轻量级框架组合。

**Spring 框架**是一个开源的 Java 应用程序框架，提供了 IOC（控制反转）和 AOP（面向切面编程）两大核心功能：
- **IOC 容器**：通过依赖注入（DI）管理对象的创建和生命周期，降低组件之间的耦合度
- **AOP 编程**：将横切关注点（如日志、事务、安全）从业务逻辑中分离，提高代码的模块化程度

**Spring MVC** 是基于 MVC 设计模式的 Web 框架，负责处理 HTTP 请求和响应：
- **DispatcherServlet**：前端控制器，统一接收所有请求
- **HandlerMapping**：请求映射，将 URL 映射到对应的 Controller 方法
- **ViewResolver**：视图解析器，将逻辑视图名解析为实际的视图资源

**MyBatis** 是一个持久层框架，支持自定义 SQL、存储过程和高级映射：
- 通过 XML 或注解配置 SQL 语句
- 提供 ORM（对象关系映射）功能，简化数据库操作
- 支持动态 SQL，灵活处理复杂查询条件

### 1.1.2 B/S 架构原理

B/S（Browser/Server）架构是一种基于浏览器和服务器的网络应用架构：
- 客户端只需要浏览器即可访问系统，无需安装专门的客户端软件
- 业务逻辑集中在服务器端，便于统一维护和更新
- 采用 HTTP 协议进行通信，具有良好的跨平台性

### 1.1.3 MVC 设计模式

MVC（Model-View-Controller）是一种软件架构模式：
- **Model（模型）**：负责数据和业务逻辑，包括实体类、DAO 层、Service 层
- **View（视图）**：负责数据展示，本项目使用 JSP 技术
- **Controller（控制器）**：负责接收请求、调用业务逻辑、返回响应

# 二、实验操作部分

## 2.1 需求分析

### 2.1.1 项目背景

随着高校学生群体的扩大，校园内产生了大量闲置物品，如旧教材、电子产品、生活用品等。传统的二手交易方式（如贴吧、微信群）存在以下问题：
- 信息分散，难以搜索和筛选
- 缺乏分类管理，浏览效率低
- 交易过程不规范，缺乏记录
- 安全性差，容易产生纠纷

因此，需要开发一个专门面向校园的二手交易平台，为学生提供便捷、安全的闲置物品交易渠道。

## 2.3 数据库设计

### 2.3.1 概念设计（E-R 图描述）

系统包含四个核心实体：

1. **用户（User）**：系统的使用者，包括普通用户和管理员
2. **商品（Product）**：用户发布的二手物品
3. **订单（Order）**：用户购买商品产生的交易记录
4. **分类（Category）**：商品的分类标签

**实体关系：**
- 用户 ←1:N→ 商品（一个用户可以发布多个商品）
- 用户 ←1:N→ 订单（一个用户可以有多个订单）
- 商品 ←1:1→ 订单（一个商品对应一个订单）
- 分类 ←1:N→ 商品（一个分类下有多个商品）

### 2.3.3 数据库设计说明

4. **索引优化**：为常用查询字段建立索引，提高查询效率

```java
public class CaptchaUtil {
    private static final String CHARS = "23456789ABCDEFGHJKLMNPQRSTUVWXYZ";
    private static final int WIDTH = 120;
    private static final int HEIGHT = 40;
    
    public static String[] generateCaptcha() {
        BufferedImage image = new BufferedImage(WIDTH, HEIGHT, BufferedImage.TYPE_INT_RGB);
        Graphics2D g = image.createGraphics();
        
        // 填充背景
        g.setColor(Color.WHITE);
        g.fillRect(0, 0, WIDTH, HEIGHT);
        
        // 生成随机验证码
        StringBuilder code = new StringBuilder();
        Random random = new Random();
        for (int i = 0; i < 4; i++) {
            char c = CHARS.charAt(random.nextInt(CHARS.length()));
            code.append(c);
            
            // 随机颜色和字体
            g.setColor(new Color(random.nextInt(150), random.nextInt(150), random.nextInt(150)));
            g.setFont(new Font("Arial", Font.BOLD, 28));
            
            // 随机旋转角度
            double theta = Math.toRadians(random.nextInt(40) - 20);
            g.rotate(theta, 25 + i * 25, 25);
            g.drawString(String.valueOf(c), 15 + i * 25, 30);
            g.rotate(-theta, 25 + i * 25, 25);
        }
        
        // 添加干扰线
        for (int i = 0; i < 5; i++) {
            g.setColor(new Color(random.nextInt(200), random.nextInt(200), random.nextInt(200)));
            g.drawLine(random.nextInt(WIDTH), random.nextInt(HEIGHT),
                       random.nextInt(WIDTH), random.nextInt(HEIGHT));
        }
        
        g.dispose();
        return new String[]{code.toString(), imageToBase64(image)};
    }
}
```

```java
private boolean validateCaptcha(String inputCaptcha, HttpSession session) {
    String sessionCaptcha = (String) session.getAttribute("captcha");
    Long captchaTime = (Long) session.getAttribute("captchaTime");
    
    // 验证码不存在或已过期（5分钟）
    if (sessionCaptcha == null || captchaTime == null ||
        System.currentTimeMillis() - captchaTime > 5 * 60 * 1000) {
        return false;
    }
    
    // 验证成功后清除（一次性使用）
    boolean valid = sessionCaptcha.equalsIgnoreCase(inputCaptcha.trim());
    if (valid) {
        session.removeAttribute("captcha");
        session.removeAttribute("captchaTime");
    }
    return valid;
}
```

```java
@Service
public class OrderServiceImpl implements OrderService {

    @Autowired
    private OrderMapper orderMapper;
    
    @Autowired
    private ProductMapper productMapper;

    @Override
    @Transactional(rollbackFor = Exception.class)  // 声明式事务
    public boolean createOrder(Order order) {
        // 1. 检查商品状态（防止超卖）
        Product product = productMapper.findById(order.getProductId());
        if (product == null || product.getStatus() != 0) {
            throw new RuntimeException("商品不存在或已售出");
        }
        
        // 2. 生成订单号
        order.setOrderNo(UUID.randomUUID().toString().replace("-", ""));
        order.setStatus(0); // 待处理
        
        // 3. 创建订单
        int result = orderMapper.insert(order);
        
        // 4. 更新商品状态为已售出
        if (result > 0) {
            productMapper.updateStatus(order.getProductId(), 1);
        }
        
        // 以上操作在同一事务中，任一失败则全部回滚
        return result > 0;
    }
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public boolean cancelOrder(Integer orderId, Integer sellerId) {
        Order order = orderMapper.findById(orderId);
        
        // 验证权限
        Product product = productMapper.findById(order.getProductId());
        if (!product.getUserId().equals(sellerId)) {
            throw new RuntimeException("无权操作此订单");
        }
        
        // 更新订单状态为已取消
        int result = orderMapper.updateStatus(orderId, 2);
        
        // 恢复商品状态为在售
        if (result > 0) {
            productMapper.updateStatus(order.getProductId(), 0);
        }
        
        return result > 0;
    }
}
```

```java
@Aspect
@Component
public class LogAspect {
    
    private static final Logger logger = LoggerFactory.getLogger(LogAspect.class);

    @Pointcut("@annotation(com.campus.annotation.Log)")
    public void logPointCut() {}

    @Around("logPointCut()")
    public Object around(ProceedingJoinPoint point) throws Throwable {
        long startTime = System.currentTimeMillis();
        
        // 执行目标方法
        Object result = point.proceed();
        
        // 记录日志
        long endTime = System.currentTimeMillis();
        recordLog(point, endTime - startTime);
        
        return result;
    }

    private void recordLog(ProceedingJoinPoint point, long time) {
        MethodSignature signature = (MethodSignature) point.getSignature();
        Method method = signature.getMethod();
        
        // 获取注解值
        Log logAnnotation = method.getAnnotation(Log.class);
        String operation = logAnnotation.value();
        
        // 获取请求信息
        ServletRequestAttributes attributes = 
            (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
        HttpServletRequest request = attributes.getRequest();
        HttpSession session = request.getSession();
        
        // 获取用户信息
        User user = (User) session.getAttribute("user");
        String username = user != null ? user.getUsername() : "匿名用户";
        String ip = request.getRemoteAddr();
        
        // 获取方法信息
        String className = point.getTarget().getClass().getSimpleName();
        String methodName = method.getName();
        
        logger.info("【操作日志】用户: {}, 操作: {}, 方法: {}.{}(), 耗时: {}ms, IP: {}",
                    username, operation, className, methodName, time, ip);
    }
}
```

#### 2.4.3.4 商品推荐算法

**RecommendServiceImpl.java - 基于内容的推荐：**

```java
@Service
public class RecommendServiceImpl implements RecommendService {

    @Autowired
    private ProductMapper productMapper;
    
    // 用户浏览历史缓存（线程安全）
    private ConcurrentHashMap<Integer, LinkedList<Integer>> browseHistoryCache = 
        new ConcurrentHashMap<>();
    
    private static final int MAX_HISTORY_SIZE = 50;  // LRU 限制

    @Override
    public List<Product> getSimilarProducts(Integer categoryId, Integer excludeId, Integer limit) {
        // 查询同分类商品
        List<Product> products = productMapper.findList(null, categoryId, 0);
        
        // 排除当前商品，按浏览量排序
        return products.stream()
            .filter(p -> !p.getId().equals(excludeId))
            .sorted((a, b) -> b.getViewCount() - a.getViewCount())
            .limit(limit)
            .collect(Collectors.toList());
    }

    @Override
    public void addBrowsingHistory(Integer userId, Integer productId) {
        browseHistoryCache.compute(userId, (key, history) -> {
            if (history == null) {
                history = new LinkedList<>();
            }
            // 移除已存在的记录（放到最前面）
            history.removeIf(id -> id.equals(productId));
            history.addFirst(productId);
            
            // LRU 策略：限制历史记录数量
            while (history.size() > MAX_HISTORY_SIZE) {
                history.removeLast();
            }
            return history;
        });
    }

    @Override
    public List<Product> getBrowsingHistory(Integer userId, Integer limit) {
        LinkedList<Integer> history = browseHistoryCache.get(userId);
        if (history == null || history.isEmpty()) {
            return Collections.emptyList();
        }
        
        return history.stream()
            .limit(limit)
            .map(productMapper::findById)
            .filter(Objects::nonNull)
            .collect(Collectors.toList());
    }
}
```

| 事务不生效 | Service 类未被 Spring 管理 | 检查 @Service 注解和包扫描配置 |
| 验证码过期 | Session 超时设置过短 | 延长验证码有效期至 5 分钟 |
